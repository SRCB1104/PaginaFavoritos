# Pagina de Favoritos

Pagina web que permite guardar perfumes como favoritos.

## Uso de la p√°gina
- El usuario puede marcar perfumes como favoritos.
- Los favoritos se guardan en el navegador.
- Al recargar la pagina, los favoritos se mantienen.
- El usuario puede quitar perfumes de la lista de favoritos.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Notas
La funcionalidad de favoritos se implementa usando JavaScript y almacenamiento local del navegador (localStorage).
